Kaplan

About the project:

Kaplan is a computer assisted translation tool project by a translator (hoping to use the plural form in the future) for fellow translators.

The project consists of kaplanpy, which is a Python library, and Kaplan Desktop, which has a web-based UI built with Electronjs and a built-in Django server for kaplanpy.

File types:

  • .docx
  • .odp
  • .ods
  • .odt
  • .txt
  • .xliff (very limited coverage)

Future plans:

Short term plans include file coverage for .xlsx, .pptx, .tmx, and .po files, as well as the ability to create/open project packages.

Longer term plans include file coverage for .sdlxliff and .sdltm files.

Final notes:

This started out as a hobby project and has lead to me getting to know wonderfully interesting people over time, so I will invest as much time as I can in this project.

I would also like to point out that both kaplanpy and Kaplan Desktop are distributed under MIT License, meaning you are free to use them commercially. If you'd like to build a better CAT tool than Kaplan Desktop using kaplanpy, go for it.
